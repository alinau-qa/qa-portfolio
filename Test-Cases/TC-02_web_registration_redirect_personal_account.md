# TC-02: Successful registration redirect to personal account (Web App)

**Priority:** High

## Requirement

In the web application, after successful registration of a new user, the user must be automatically authenticated and redirected to the personal account page.

---

## Preconditions

- The user is not registered in the system  
- The email is not registered in the system  

---

## Steps

### Step 1
**Action:** Open the URL page  
**Expected result:** The login page is displayed  

---

### Step 2
**Action:** Click the "Register" button  
**Expected result:** The registration page is opened  

---

### Step 3
**Action:** Enter a valid email in the "Enter email" field  
**Expected result:** The email is displayed in the field  

---

### Step 4
**Action:** Enter a password in the "Enter password" field  
**Expected result:** The password is displayed in the field  

---

### Step 5
**Action:** Enter the same password in the "Confirm password" field  
**Expected result:** The password is displayed in the field  

---

### Step 6
**Action:** Click the "Register" button  
**Expected result:** The page for entering the email verification code is opened  

---

### Step 7
**Action:** Go to the email inbox used for registration and copy the verification code  
**Expected result:** The code is copied  

---

### Step 8
**Action:** Enter the code in the field "Enter the code sent to your email"  
**Expected result:** The code is displayed in the field  

---

### Step 9
**Action:** Click the "Confirm" button  
**Expected result:** A message appears: "User successfully registered."  

---

### Step 10
**Action:** Click the "Return to homepage" button  
**Expected result:** The login page is opened  

---

### Step 11
**Action:** Enter the registered email in the "Enter email" field  
**Expected result:** The email is displayed in the field  

---

### Step 12
**Action:** Enter the password in the "Enter password" field  
**Expected result:** The password is displayed in the field  

---

### Step 13
**Action:** Click the "Login" button  
**Expected result:** The personal account page is opened
