# Test Cases

This folder contains manual test cases for the web application.

## What is tested

Main features of the app:

### User Profile
- Upload avatar
- Update profile settings

### Products
- View product details
- Add product to Favorites (❤ button)

---

## Test Case Format

Each test case is written step by step.

### Example

### Step 1
**Action:** Open profile page  
**Expected result:** Profile page is displayed  

### Step 2
**Action:** Navigate to "Settings"  
**Expected result:** Settings page is opened  

### Step 3
**Action:** Click "Upload avatar"  
**Expected result:** File picker opens  

### Step 4
**Action:** Select image file  
**Expected result:** Avatar is updated successfully
