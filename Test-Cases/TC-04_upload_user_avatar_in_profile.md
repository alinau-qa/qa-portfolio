# TC-04: Upload user avatar in profile (Max size 5MB)

**Priority:** Medium

---

## Requirement
In the web application, a user can upload an avatar in their personal profile using a local image file. The maximum allowed file size is 5MB.

---

## Preconditions
- The user is authorized (logged in)
- A valid image file exists (`avatar_5Mb.jpeg`, size ≤ 5MB)

---

## Steps

### Step 1
**Action:** Open "My Profile" section  
**Expected result:** Profile page is displayed

---

### Step 2
**Action:** Navigate to "Settings"  
**Expected result:** Settings page is opened

---

### Step 3
**Action:** Click "Upload Avatar" button  
**Expected result:** File explorer is opened

---

### Step 4
**Action:** Select an avatar image file (≤ 5MB)  
**Expected result:** Selected file is highlighted

---

### Step 5
**Action:** Click "Open"  
**Expected result:** Avatar preview is updated with selected image

---

### Step 6
**Action:** Click "Save"  
**Expected result:** Changes are saved successfully
